# Quick Start Guide

## 🚀 Get Started in 30 Seconds

### 1. Open Terminal/Command Prompt
```bash
cd python_app
```

### 2. Run the App
```bash
python main.py
```

### 3. That's it! 🎉
The Formula Calculator window opens automatically.

---

## 📖 First Time Users

### What to Do:
1. **Open the app** using command above
2. **Click a formula** from the left list (try "Circle Area")
3. **Enter a number** (try 5)
4. **Click Calculate** 
5. **See the result!** (78.54 sq units)

### Try These First:
- 🌡️ Celsius to Fahrenheit (enter 25)
- 📐 Pythagorean Theorem (enter 3 and 4)
- 👥 Body Mass Index (enter 70 and 1.75)

## 🐛 Troubleshooting

### App won't start?
```bash
python --version  # Should be 3.6+
python -m tkinter  # Tests GUI (should open window)
```

### "All inputs must be valid numbers"?
- Use decimal point: 3.5 (not 3,5)
- No units: enter 5 (not 5m)
- All fields required

### Formula error?
- Read the formula description
- Check your values are reasonable
- Try the Help button

---
